 # Project 1 : Lending Case Finance Study Analysis

The main objective of the project is to identify the risky loan applicants, thus further extracting the relevant insights from the dataset having  111 no of independent features in order to reduce the risk factor of lending the loan to the not eligible candidate. 

## Steps considered 
  
1. Handling the invalid features
2. Handling the missing values
3. Handling of categorical value(labelencoding)
3. Univariate analysis
4. Bi-variate analysis 

## Technologies Used
Python - 3.10
Python  library -Pandas, matplotlib, seaborn, label encoder
Python editor - Jupyter notebook.
